Om beräkningen
Analysen bygger på en förenklad schablonmodell enligt lagförslaget. Elpris behandlas som ett årsmedelvärde och produktion anges som antagen nettoårsproduktion per verk. Resultatet ska tolkas som ett scenario, inte som ett faktiskt utfall eller en prognos.
Bostadens unika Objekt-ID
Objekt-ID är ett unikt identifieringsnummer för varje bostad. Det tilldelas när bostaden registreras och ändras därefter inte. ID:t kommer från Lantmäteriets databas och används för att entydigt identifiera bostaden.
